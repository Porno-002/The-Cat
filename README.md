## 项目说明：猫咪主题网页封面

这个小项目是根据 Figma 设计稿「铃木杏子-s team library / 封面」做的一个静态网页。

### 页面文件结构

- **根目录**
  - `index.html`：项目入口文件，会自动跳转到真正的首页页面。
- **pages 文件夹**
  - `pages/homepage.html`：首页（封面）对应的 HTML。
- **styles 文件夹**
  - `styles/homepage.css`：首页（封面）对应的 CSS。

你只需要：

1. 用浏览器打开根目录下的 `index.html`
2. 浏览器会自动跳到 `pages/homepage.html`
3. 就可以看到和 Figma 类似的封面页面（欢迎标题、顶部导航、按钮等）

### 布局说明（简化版）

- **整体尺寸**：按照 Figma 画布，页面内容区域是 1920 × 1080（电脑端设计）。
- **顶部导航条**（Guide）
  - 左侧灰色顶部条作为背景。
  - 右上角有「帮助」「登录」两个小按钮。
  - 中间有一条深灰色导航条，包含：
    - 首页
    - 猫咪科普
    - 行为指南
    - 养猫日志
    - 猫咪美图
    - 救助中心
    - 未来规划
    - 反馈建议
    - 关于网站
- **主内容区域**
  - 左上是一个大标题 `Welcome!`
  - 右侧是一组「卡片按钮」文字：
    - 看看小猫！→
    - 了解小猫！→
    - 饲养指南！→

### 如何修改

- 想改文字：在 `index.html` 里找到对应的中文或英文文本，直接修改即可。
- 想改颜色或大小：在 `style.css` 里找到对应的类名（例如 `.top-nav`、`.hero-title`），调整颜色、字号等。

如果你告诉我「想改哪里」，我可以继续帮你一步步完善页面。
